# lean-mockingbird

This repository contains Lean statements and proofs for [Raymond M. Smullyan](https://en.wikipedia.org/wiki/Raymond_Smullyan)'s book [*To Mock a Mockingbird and Other Logic Puzzles: Including an Amazing Adventure in Combinatory Logic*](https://en.wikipedia.org/wiki/To_Mock_a_Mockingbird), Chapters 9 to 23.
